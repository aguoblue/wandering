# 点击地图设置终点功能测试指南

## 功能说明
实现了点击地图任意位置作为终点的功能，具体包括：
1. 点击地图上的任意位置
2. 自动使用逆地理编码获取该位置的地址信息
3. 将地址名称填充到搜索框中
4. 在地图上添加一个标记点
5. 点击标记点可以查看详细信息窗口

## 使用步骤
1. 确保已配置高德地图 API 密钥（VITE_AMAP_KEY）
2. 启动开发服务器：`npm run dev`
3. 在地图上点击任意位置
4. 观察搜索框是否自动填充地址名称
5. 查看地图上是否出现了新的标记点
6. 点击标记点，确认信息窗口显示正确

## 技术实现
- 使用 `AMap.Geocoder` 进行逆地理编码
- 通过 `mapRef.current.on('click', handleMapClick)` 监听地图点击事件
- 使用全局方法 `window.updateSearchBoxInput` 更新搜索框值
- 添加了动画效果的标记点和信息窗口

## 注意事项
- 需要有效的网络连接才能使用逆地理编码功能
- 第一次点击可能需要等待 API 加载完成
- 标记点会覆盖之前的搜索结果，需要重新选择交通方式规划路线
